# Handoff

## 中文

这是一个用于生成 `Handoff.md` 交接文档的 skill。

用途：
- 会话结束前整理交接内容
- 记录正在做什么、已完成什么、卡在哪里、下一步做什么
- 强调绝对不能踩的坑

使用时：
- 先提供主名
- 如需保存为文件，也提供保存位置
- 其余内容会按交接模板生成

## English

This skill generates a complete `Handoff.md` transfer document.

Use it to:
- wrap up a session
- capture what the work is, what is done, what is blocked, and what comes next
- record critical pitfalls to avoid

When using it:
- provide the main name first
- provide a save location if you want it written to a file
- the rest follows the handoff template
